Install the UniversalLogg agent

The agent runs on your Linux server, watches your log files, and automatically ships new events to UniversalLogg every 5 seconds. No manual uploads needed.

Step 1

Generate an API key

Go to Settings and generate an API key. You'll need it in the next step.

Go to Settings
Step 2

Download the agent on your server

SSH into your Linux server and run:

curl -O https://security-event-dashboard-production.up.railway.app/agent/download
mv download UniversalLogg-agent.py
Step 3

Install the dependency

pip3 install requests
Step 4

Run the agent

Replace YOUR_API_KEY with the key from Step 1:

python3 UniversalLogg-agent.py --key YOUR_API_KEY

The agent will start watching your log files and shipping events automatically.

Step 5 — Optional

Install as a background service

To run the agent automatically on boot (recommended for production):

sudo python3 UniversalLogg-agent.py --key YOUR_API_KEY --install

This installs a systemd service that starts automatically and restarts if it crashes.

Which log files does the agent watch?
By default it watches /var/log/auth.log, /var/log/syslog, and common web server logs. To watch custom files, use the --files flag:

python3 UniversalLogg-agent.py --key YOUR_KEY --files /var/log/myapp.log /var/log/auth.log
