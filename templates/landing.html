Features

Everything you need to stay secure

Log file ingest

Upload .log or .txt files directly from the dashboard. Events are classified and stored instantly.

Real-time email alerts

Get notified instantly when a high-severity event is detected — brute force attempts, port scans, and more.

API ingest

Push log lines directly from your servers using a Bearer token. No file upload needed in production.

Multi-tenant

Each organisation's data is fully isolated. Invite your team and collaborate on the same dashboard.

Usage dashboard

See exactly how many events you've ingested this month and your projected bill — no surprises.

Usage-based billing

Free up to 500 events/month. Above that, pay only for what you use via Stripe — no fixed monthly fee.
